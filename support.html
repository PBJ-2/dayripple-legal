결제가 완료되지 않았을 때

앱의 설정 → 결제 상태 → 미완료 구매 확인을 눌러 주세요. 같은 스토어 거래를 다른 공간에 중복 적용할 수는 없습니다.

계정 삭제

앱의 설정 → 계정 → 계정 삭제에서 바로 삭제하거나, 계정 삭제 요청 페이지를 이용할 수 있습니다.

When a payment didn’t go through

In the app, tap Settings → Payment status → Check unfinished purchases. The same store transaction cannot be applied to more than one space.

Account deletion

Delete directly in the app under Settings → Account → Delete account, or use the account deletion request page.
